### General Info
***
Before you start, dont forget to choose your mobile device emulator.
It may take a few seconds up to one minute to start the project, so please be patient.

Introduction for the installation of the Packages:
- Open "Visual Studio"
- Go to "Tools"
- Go to "NuGet Package Manager"
- Open "Manage NuGet Packages Manager for Solution..."
Now you have to "Browse" the necessary packages below, and download it.

Website to scan the QR-Code: 
https://zxing.appspot.com/generator
